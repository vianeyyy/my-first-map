# Places I've Explored

![Map](img/Interesting_Places!.png)
## About This Project
["My Interactive Map"]() is an interactive web map showcasing locations I've visited throughout the Bay Area. Using custom markers and detailed popups, this map serves as both a personal travel journal and an educational resource for anyone interested in web mapping technologies.

## Live Demo
Explore the map in action:

## Project Purpose
This map was created to:
- Document and share my personal travel experiences visually
- Demonstrate practical applications of interactive web mapping
- Provide open-source code that others can learn from and adapt
- Combine technical development with personal storytelling

## Technical Implementation
Built with modern web technologies:
- **Mapping**: [Mapbox GL JS](https://docs.mapbox.com/mapbox-gl-js/guides) for interactive cartography
- **Data Structure**: GeoJSON for organizing location information
- **Development**: [Visual Studio Code](https://code.visualstudio.com/) with Live Server extension
- **Hosting**: [GitHub Pages](https://pages.github.com/) for public accessibility
- **Design**: Custom color scheme and visual styling
